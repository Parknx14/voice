# Voice

#### How to Use:
1. Go to [link](https://devblin.github.io/voice/) to play with it.
2. Click on `start` to start mic.
3. Clicking on `stop` switch-offs the mic.

#### Commands (Speech):
* Start - To move at start position.
* Left - To move left.
* Right - To move right.
* Down - To move down.
* Up - To move up.
